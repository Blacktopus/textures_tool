Инструкция пользователя:
- сохраните исходные текстуры для бэкапа
- необходимо понимать, что программа создаёт новые файлы с нумерацией поверх изображения, убрать нумерацию невозможно
- программа разработана, чтобы тестеры могли оперативно скидывать переводчикам скриншоты текстур, а переводчики оперативно находить их в файлах игры
- необходимо опреелить одного ответственного за хранение исходных файлов, файлов с переводом, и csv файла, чтобы не возникало путаницы при различной нумерации у разных пользователей

- создайте папку для текстур с нуммерацией
- укажите путь с исходными текстурами
- создайте папку для текстур с нуммерацией и укажите путь к ней
- дождитесь создания всех файлов в конечной папке
- в конечной папке вы можете найти файл csv для удобного поиска номера текстуры из игры в папке по имени
